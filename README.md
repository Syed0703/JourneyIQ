Node.js (v18 or later recommended)

## 1. Clone the Repository
Bash

git clone https://github.com/Syed0703/JourneyIQ.git
cd ai-travel-planner

## 2. Install Dependencies
Bash

npm install
# or
# yarn install

## 3. Run the Development Server
Bash

## 4. Build for Production
Bash

npm run build
npm run start
